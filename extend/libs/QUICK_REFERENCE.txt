HELPERS.PY - Конвертирование:
  from plugins import helpers
  helpers.to_int("42")           # 42
  helpers.to_bool("true")        # True
  helpers.to_camel_case("hello_world")  # helloWorld

HELPERS.PY - Валидация:
  from plugins import helpers
  helpers.is_email("user@example.com")   # True
  helpers.is_phone("+380961234567")      # True
  helpers.has_length("password", 8)      # True/False

HELPERS.PY - Файлы:
  from plugins import helpers
  helpers.write_file("test.txt", "Hello")
  content = helpers.read_file("test.txt")
  size = helpers.file_size("test.txt")

DSL - Простой скрипт:
  from plugins.dsl_integration import create_dsl_system
  context, executor = create_dsl_system()
  executor.execute_script("""
    set(name, "Иван")
    print("Привет", $name)
  """)

DSL - Builder:
  from plugins.dsl_integration import DSLBuilder
  script = DSLBuilder() \\
    .set_var("x", 10) \\
    .print_msg("x =", "$x") \\
    .build()

Q: Нужно ли устанавливать пакеты для helpers.py?
A: Нет! Только Python stdlib. DSL тоже работает без дополнительных пакетов.

Q: Можно ли использовать helpers.py отдельно?
A: Да! Он полностью независим от других модулей.

Q: Как использовать DSL для своего проекта?
A: Импортируйте create_dsl_system() и пишите скрипты.

Q: Где примеры использования?
A: В examples_qol.py (20 примеров) и в документации.

Q: Поддерживается ли украинский язык?
A: Да! Вся документация на украинском.

ОШИБКА: File not found

РЕШЕНИЕ: Проверьте что переменные установлены через context.set_var()
